Predicting College Football Wins

Using the data set College Football Statistics from Matt Hixon, my partner, Justin Thoms, and I ran basic machine learning algorithms to try and discover which specific metrics help predict which team is most likely to be successful on the college level. Testing metrics like Yards For, Yards Against, Turnovers, and Home Field Advantage, we ran tests to see how often each differential predicted the result of the game, using R.

After running our simulations, we found that the best predictor was turnover differential, by a mile, showing truly how important holding onto the ball is for keeping up against opposing teams.
